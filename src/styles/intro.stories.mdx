# CSS

This library also includes a compiled master CSS stylesheet that contains all of the main CSS classes etc from the ONS website and design system.

### Importing our main stylesheet

Import the CSS directly within your top-level `+layout.svelte` file.

```html
<!-- src/routes/+layout.svelte -->
<script>
  import "@onsvisual/svelte-components/dist/css/main.css";
</script>

<slot />
```
